# 概述





### 分组活动(5min)

目标:建立学员链接，分组
主要方式:学员系统思考自评分围圈报数(4-5组)

### 小组活动(5分钟)

目标: 破冰开脑，链接系统定义
讲师点评要点:

* 要素
* 相互关联

**同时带出系统的三个特性**

* 层次行
* 适应性
* 自组织行(稳定趋势性)

### 小组讨论(15分钟）
目的:为什么要系统思考?什么系统思考？
准备材料:国王的抉择

点评点:
* 容易的办法往往无功而返
* 最小的变化产生意想不到的效果(杠杆往往被忽视)
* 越使劲反弹越大
* 系统思考的特性
    1. 整体
    2. 相关
    3. 动态
    4. 复杂
